# Revere Point Count

The Revere Point Count is a level 2 counting systems from the book Playing Blackjack as a Business.

| Attribute             | Value         |
| ---                   | :-:           |
| Balanced Count        | Y             |
| Side Counts           | Y (Aces)      |
| True Count Divisor    | 1 Deck        |
| Take Insurance at     | +2 TC if no surplus of Aces |
